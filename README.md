# Generalized Linear Model (GLM) Analysis in R

This project demonstrates the application of Generalized Linear Models (GLM) using R for statistical modeling and data analysis.

The goal of the project is to explore relationships between variables and evaluate model performance using appropriate link functions and distributions.

## Methods

The analysis includes:

- Data exploration and preprocessing
- Fitting Generalized Linear Models (GLM)
- Model interpretation and diagnostics
- Visualization of model results

## Technologies

- R
- RStudio
- GLM (Generalized Linear Models)
- ggplot2

## Files

- `final project.Rproj` – main script for fitting and evaluating the GLM model  
- `APPENC10.csv` – dataset used in the analysis  
- `presentation.pptx` – a summary of the test and comparison between different models  

## Key Concepts

- Generalized Linear Models (GLM)
- Regression analysis
- Statistical inference
- Data visualization

## Purpose

This project was created as part of graduate-level coursework in Applied Mathematics and Statistics at Johns Hopkins University. It demonstrates practical experience with statistical modeling and data analysis using R.
